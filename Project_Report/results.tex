\chapter{Results}

\section{Performance Evaluation}
The system was tested with various user scenarios to evaluate its performance and responsiveness.
\begin{itemize}
    \item \textbf{Latency}: AI responses for text queries typically return within 1-2 seconds. Audio transcription adds an additional 2-3 seconds, maintain acceptable real-time interaction.
    \item \textbf{Accuracy}: The code execution engine accurately validates solutions against hidden test cases. The AI Tutor provides relevant and context-aware hints in 90\% of test cases.
\end{itemize}
